## Spec Instructions

When creating the spec for 017-vue-frontend, use these documents as the baseline:

1. Load the design document (`research/017-vue-frontend-design.md`) during `/speckit.specify`
2. The mockup is the source of truth for layout, navigation, interactions, and confirmation flows
3. The component mapping in the design doc is a suggestion — adjust based on implementation needs
4. Deferred items (backup policies backend, trait adapters) are listed at the bottom of the design doc and tracked as GitHub issues
