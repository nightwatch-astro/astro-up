## Task

Add `check_survey_eligible()` function that combines operation count >= threshold AND completed_at is None AND (dismissed_at is None OR now - dismissed_at > 30 days).

## File

`crates/astro-up-core/src/config/mod.rs`

## Acceptance Criteria

- Returns `bool`
- Takes `Connection` (for count query) and `UiConfig` (for thresholds/timestamps)
- Returns `true` only when all three conditions met
- Returns `false` when below threshold, completed, or within snooze window
