## Task

Integrate `SurveyDialog` into Dashboard — check eligibility on mount, show dialog if eligible.

## File

`frontend/src/views/DashboardView.vue`

## Behavior

- `onMounted`: invoke `check_survey_eligible` Tauri command
- If returns `true`: set `showSurvey` ref to `true`
- Mount `<SurveyDialog :visible="showSurvey" @update:visible="showSurvey = $event" />`
- Check runs once per mount, not on re-render
